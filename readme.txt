=== BP-TagBox ===
Contributors: Yaway Media
Tags: Buddypress, Tagbox, Social, Bookmarks
Requires at least: WPMU 2.7.1, BuddyPress 1.0.2
Tested up to: WPMU 2.8.4, BuddyPress 1.1
Stable tag: 1.0

Add bookmarks from the german social bookmarks service TagBox to the profiles of a BuddyPress installation.

== Description ==

English: Add bookmarks from the german social bookmarks service <a href="http://www.tagbox.de" target="_blank" title="Social Bookmarks Service TagBox">TagBox</a> to the profiles of a BuddyPress installation.

German: BP-TagBox erweitert den Profilbereich der BuddyPress-User um die M&ouml;glichkeit, die beim <a href="http://www.tagbox.de" target="_blank" title="Social Bookmarks Dienst TagBox">Social Bookmarks Dienst TagBox</a> abgespeicherten Favoriten zu importieren. In den Einstellungen des Nutzerprofils hat man die M&ouml;glichkeit, TagBox-Benutzernamen und die gew&uuml;nschte Anzahl der Favoriten zu definieren.

== Installation ==
= English Version: =
1. Upload this plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done!

= German Version: =
1. Pluginordner in das Verzeichnis `/wp-content/plugins/` hochladen.
2. Plugin via Adminpanel aktivieren.
3. Fertig!

== Frequently Asked Questions ==

Yes it is! Because the TagBox Service is only in German.
